# AIM Research Team OS

논문 작성과 HUFStudy 운영을 위한 내부 팀 OS입니다. 고객용 AdTrace AI 서비스와 분리된 별도 정적 앱입니다.

## 목적

- 데일리 스크럼 기록
- 태스크 상태와 마감일 관리
- 논문 섹션별 작성 상태 추적
- A/B 테스트 설계와 CTR, CVR, 구매버튼 클릭 측정 방식 정리
- 주간보고서, 선행연구, 광고 자극물, 회의록 아카이빙
- 고객검증 인사이트와 다음 액션 누적

## 실행

브라우저에서 `index.html`을 열면 바로 실행됩니다.

개발 서버로 확인하려면:

```bash
python3 -m http.server 5174
```

접속:

```text
http://127.0.0.1:5174
```

## 저장 방식

서버, DB, AI API 없이 브라우저 `localStorage`에 저장됩니다. 상단의 `JSON 내보내기` 버튼으로 현재 데이터를 백업할 수 있습니다.

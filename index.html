연구문제 재정의 및 역할 분담

AI-like score가 소비자 신뢰, CTR, CVR, 구매버튼 클릭, ROI 판단에 미치는 영향을 검증하는 방향으로 연구문제를 재정의했습니다.

  • 최서린: 문헌·보고서·논문 구조
  • 지우진: A/B 테스트 및 행동지표 분석
  • 최은서: 광고 이미지·risk band·hold-out set

A/B 테스트 설계

AI 광고와 인간 제작 광고의 행동지표 차이를 검증합니다.

측정 지표 정의

CTR·CVR·구매버튼 클릭을 같은 기준으로 기록합니다.

CTR

광고 클릭 수 / 광고 노출 수. AI-like score 구간별 클릭 반응 차이를 봅니다.

CVR

전환 수 / 클릭 수. 클릭 이후 구매 의향 또는 신청 행동으로 이어지는지 봅니다.

구매버튼 클릭

상세 페이지 내 구매 CTA 클릭 이벤트. 실제 구매 전 단계의 강한 행동 신호로 기록합니다.
